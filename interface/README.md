# Pre-launch waitlist page

![JARS Landing Preview](src/components/images/Screenshot%202026-01-11%20170941.png)

This repository hosts the static frontend for the JARS marketing and waitlist layer. It is decoupled from the core execution engine to ensure high availability and separate concerns.

## Tech Stacks

* **Framework:** Next.js 14 (App Router)
* **Styling:** Tailwind CSS
* **Database:** PostgreSQL
* **Deployment:** HTML Export (`output: export`)

## Local Development

### Prerequisites
* Node.js 18+
* npm
